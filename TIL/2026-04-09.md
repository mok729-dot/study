3. 데이터 전처리

import pandas as pd

train = pd.read_csv('train.csv')
test  = pd.read_csv('test.csv')

# 결측치 처리 (Age 평균 대체)
age_mean = train['Age'].mean()
train['Age'] = train['Age'].fillna(age_mean)
test['Age']  = test['Age'].fillna(age_mean)

# 성별 인코딩 (문자 → 숫자)
train['Sex'] = train['Sex'].map({'male': 0, 'female': 1})
test['Sex']  = test['Sex'].map({'male': 0, 'female': 1})

# 독립변수 / 종속변수 분리
train_x = train[['Pclass', 'SibSp', 'Age', 'Sex']]
train_y = train['Survived']
test_x  = test[['Pclass', 'SibSp', 'Age', 'Sex']]
⚠️ 핵심 원칙
test의 결측치도 train의 평균으로 채운다 → 데이터 누수(leakage) 방지
머신러닝 모델은 문자열을 직접 학습 못 하므로 반드시 숫자 인코딩
4. 베이스라인 v1 - 디시전트리

from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 학습/검증 70:30 분할
X_tr, X_val, y_tr, y_val = train_test_split(
    train_x, train_y, test_size=0.3, random_state=42
)

# 모델 학습
model = DecisionTreeClassifier(random_state=42)
model.fit(X_tr, y_tr)

# 검증 정확도
val_pred = model.predict(X_val)
print(f"Accuracy: {accuracy_score(y_val, val_pred):.4f}")

# test 예측
test_y_pred = model.predict(test_x)
왜 train을 또 70:30으로 나눠?
test.csv는 정답이 없어서 직접 점수를 못 봄
train의 30%를 떼서 가짜 test(검증용) 로 사용 → 모델 성능 미리 확인
제출 점수: 0.67066

5. 베이스라인 v2 - 랜덤포레스트 + 변수 추가

from sklearn.ensemble import RandomForestClassifier

# Fare 결측치 처리 (test에 1개 있음)
test['Fare'] = test['Fare'].fillna(test['Fare'].mean())

# 변수 6개로 확장
features = ['Pclass', 'Sex', 'Age', 'SibSp', 'Parch', 'Fare']
train_x = train[features]
test_x  = test[features]

# 모델 변경: DecisionTree → RandomForest
model = RandomForestClassifier(
    n_estimators=300,
    max_depth=6,
    random_state=42
)

# 검증
X_tr, X_val, y_tr, y_val = train_test_split(train_x, train_y, test_size=0.3, random_state=42)
model.fit(X_tr, y_tr)
print(f"검증 Accuracy: {accuracy_score(y_val, model.predict(X_val)):.4f}")

# 전체 train으로 재학습 후 예측
model.fit(train_x, train_y)
test_y_pred = model.predict(test_x)
제출 점수: 0.75615 (+0.085 ⬆)
